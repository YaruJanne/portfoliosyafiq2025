Duel Beats
Description:

A PC rhythm game using Sony Mocopi for full-body motion capture. It features a unique "Real-Time Beat Generation" system that analyzes any user-uploaded .wav song to create gameplay automatically.

Platform:

PC, Sony Mocopi

Tools Used:

Unreal Engine 5.4, MetaSound, Sony Mocopi, Adobe Audition

Development Insight:

One of the primary hurdles was achieving low-latency motion capture suitable for a rhythm game. Initial testing with vision-based systems like Radical resulted in a delay of 200–300ms, which broke the immersion needed for precise timing. To solve this, the project migrated to the Sony Mocopi inertial system, reducing latency to approximately 50ms. However, this introduced a new challenge of "sensor drift," requiring the implementation of a quick-recalibration feature to ensure the player's virtual avatar remained aligned during extended play sessions.

Another significant challenge was the accuracy of the automated beat generation system. The initial algorithm frequently generated "false positive" beats when analyzing complex electronic tracks with high noise floors. To address this, a custom frequency-gating system was built using MetaSound, which applied low-pass filters to specifically isolate kick drums and bass frequencies. This significantly improved the precision of the generated beat maps, ensuring that gameplay targets synced correctly with the song's actual rhythm rather than background noise.
